# Installation


## ?? Requirements & Installation

### ?? System Requirement
You need to have Python 3.11+ installed on your PC.

1. **Install AutoClaudeBot**
   - Inside Github [Github Page](https://github.com/smanav225-hub/AutoClaudeBot)
   - Click on the green `<> Code` button, then click **Download ZIP**.
   - Once installed, go to your Downloads folder, right-click the zip file, and select **Extract Here**.

2. **Install Requirements**
   - Open Command Prompt (CMD) or Terminal and run:

   ```bash
   pip install discord.py fastapi uvicorn httpx requests pillow
   ```

   - Wait for it to finish installing.

3. **Discord Token**
   - Watch this YouTube video from **00:00 to 07:00** (The first 7 minutes):
     - [Steps To Get a Discord Token](https://www.youtube.com/watch?v=CHbN_gB30Tw&t=336s)
   - Once you get your Discord token and follow the first 7 minutes, continue to step 4.

4. **Starting App**
   - Open the extracted folder from step 1 and find `Start.py`.
   - Run `Start.py`.
   - When the browser opens (localhost), click **Setting** in the left sidebar.
   - Paste your token into the Token entry box and click **Save**.
   - Refresh/reload the page.

5. **How to Use the App**
   - Watch this tutorial video:
     - [Tutorial Video](https://www.youtube.com)
